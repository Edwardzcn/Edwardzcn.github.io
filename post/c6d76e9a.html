【CSU-1592】解题报告（区间DP入门）

原始题目

1592: 石子归并

  • Time Limit: 1 Sec
  • Memory Limit: 128 Mb
  • Submitted: 896
  • Solved: 421

Description

现在有$n$堆石子，第$i$堆有$a_i$个石子。现在要把这些石子合并成一堆，每次只能合并相邻两个，每次合并的代价是两堆石子的总石子数。求合并所有石子的最小代价。

Input

第一行包含一个整数$T ( T \le 50) $ ，表示数据组数。

每组数据第一行包含一个整数$ n ( 2 \le n \le 100 ) $，表示石子的堆数。

第二行包含$n$个正整数$ a_i ( a_i \le 100)$，表示每堆石子的石子数。

Output

每组数据仅一行，表示最小合并代价。

Sample Input

2
4
1 2 3 4
5
3 5 2 1 4

Sample Output

19
33

Hint

Source

国防科学技术大学第十八届银河之光文化节ACM程序设计竞赛初赛

如题

解题思路

  • 区间dp

  • 状态转移方程：$dp[i][j]=min(dp[i][j],dp[i][i+k]+dp[i+k+1][j]+sum[j]-sum[i-1])$;
