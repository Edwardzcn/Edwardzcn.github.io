第四节 货币制度

我国的货币制度

  1. 我国的法定货币是人民币。
  2. 人民币是我国唯一合法流通货币。
  3. 人民币的发行实行高度集中统一于中国人民银行。
  4. 人民币的发行保证是国家拥有的商品物资，黄金外汇储备主要是作为国际支付的准备金。

纸币发行准备制度的方式：

  1. 以黄金或外汇储备准备的发行
  2. 无准备发行

第二钟以港币发行为例。
香港地方三家银行汇丰，

央行可以通过行政命令，直接宣布汇率变化，来影响

金属本位

金银复本位

  1. 平行本位：即金银币平等，同事都是本位币，都可以自由铸造和融化自由地输出入国境，金银币的交易比率由
    市场决定，国家没有法律规定。

    问题：由于任一商品都拥有两个价格，比例随金银比价波动，造成价格混乱，给交易带来了很多麻烦。

  2. 双本位：格雷欣法则，劣币驱逐良币

    劣币驱逐良币律Gresham’s Law：金银复本位制下，两种实际价值不同而含义相同的金属铸币流通时，实际价值高的会通过熔化，
    储藏或者到国外而退出流通。

  3. 跛行本位

    金币可以自由铸造，但是银币不能自由铸造，只可以无限偿还。

金本位制的三种形式

  1. 金币本位：纯粹的金币本为：
    自由铸造，自由兑换，自由输出入国境

  2. 金块本位：金币不再流通，黄金纸币不可自由兑换，一定限额。
    两次世界大战期间英法实行，黄金储备不够不足以支持自由兑换（富人本位）。

  3. 金汇兑本位：不铸造金币，市场上不流通，本币不可兑换黄金，但本币可以以固定汇率兑换成某一实行金币或金块本位的国家
    的货币    ，并在该国兑换黄金。（无法兑换，提供其他两种本位的一个固定汇率关系）
    纸币与黄金的关系进一步弱化。

第五节 国际货币体系

“黄金——美元”本位制的布雷顿森立体系，美元本位制牙买加体系。

国际货币制度的研边

国际金本位制度（1880-1913） => 浮动汇率制度（1914-1944） => 布雷顿森林体系（1945-1973） => 牙买加体系（1978）

国际金本位制

特点：

原因

  1. 黄金产量不足，长期通货紧缩
  2. 黄金产品分布不均
  3. 第一次世界大战爆发的刺激

布雷顿森林体系的建立

“双挂钩一固定” 美元同黄金挂钩，其他货币与美元挂钩。美元与黄金兑换固定（35美元1盎司）。
