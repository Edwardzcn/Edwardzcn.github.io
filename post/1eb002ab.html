The First Step

题目摘要引言

Title

Language Support for Designing Geo-replicated Services with Fast Performance and Preserved Semantics

针对高性能保留语义的跨地理服务设计的语言支持

关键词：跨地理、语义保护

Abstract

互联网服务提供商使用跨地理的服务（应用场景）在不同大陆上部署副本来提升应用的响应时间。

问题、挑战：长距离同步通信成本太大。混合一致性模型和细粒度的一致性协议提供了机遇和挑战。本文给出了一种细粒度的一致性模型、编程接口及形式化的操作语义

Introduction

回顾大规模、跨地域服务的广泛应用。

传统的 distributed storage systems 仅使用线性一致性模型 Windows Azure Storage 和 Google Spanner。

随着大规模互联网服务使用者进一步增多，提供商开始提供 geo-replicated 系统来提升响应时间，但是这种场景下，长距离通信开销变得很大，成为瓶颈问题。（见参考文献 1，Consistency tradeoffs in modern distributed databasesystem design: Cap is only part of the story）
