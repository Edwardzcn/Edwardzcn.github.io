原始题目

2183: 击鼓传花简单版

  • Time Limit: 1 Sec
  • Memory Limit: 128 Mb
  • Submitted: 165
  • Solved: 41

Description

岳麓山下旌旗卷，CSU运动会隆重开幕了，小Q所在的班级参加了“击鼓传花”趣味比赛。

比赛规则如下，$n$个队员(不包括小Q)围成一个圈，每个人手里拿着一些花束(数量也可能为0)，小Q站在圆圈的中间，队员需要传递花束给小Q，每个鼓点只允许一个人传送一束花，最终要求圆圈上所有人的花束都传递给小Q。请问需要花多少鼓点呢？

Input

多组数据。

第一行$n$，代表队员数量。

第2行到第$n+1$行，每行一个整数$a_i$，代表每个队员最开始拿的花束数量。$(0 ≤ a_i ≤ {10}^{6})$
所有组的n之和不超过 $ {10}^{3} $

Output

一个整数，代表需要的鼓点数。

Sample Input

4
1
2
5
4

Sample Output

12
